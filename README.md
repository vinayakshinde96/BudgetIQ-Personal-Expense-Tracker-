# 💰 BudgetIQ – Finance Management & Analytics Platform

BudgetIQ is a full-stack finance management application built with **Python, Flask, MySQL, HTML, CSS, and JavaScript**. The platform enables users to manage financial transactions, monitor budget performance, track spending patterns, and gain actionable insights through an interactive analytics dashboard.

## 🚀 Key Features

* Secure User Authentication
* Expense Management
* Budget Tracking & Monitoring
* Financial Analytics Dashboard
* Category-wise Expense Analysis
* Real-Time Financial Insights
* MySQL Database Integration

## 🛠️ Tech Stack

**Frontend:** HTML, CSS, JavaScript
**Backend:** Python, Flask
**Database:** MySQL
**Tools:** Git, GitHub, VS Code

## 📊 Project Highlights

* Designed a relational database for financial data management.
* Developed secure authentication and session handling.
* Implemented CRUD operations with MySQL.
* Built interactive dashboards for financial reporting and analysis.
* Applied SQL queries for data aggregation and insight generation.
